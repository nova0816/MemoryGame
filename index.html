Changelog & History

v1.1.8 (Latest) 2026-06-20
  • Increased card click zoom scale to 2x (double size) to ensure maximum readability and accessibility for children.
  • Implemented dynamic viewport centering offsets: when a card zooms, its position is calculated relative to the screen center, and it is translated 45% towards the center. This completely prevents cards on the screen borders from being cut off or trimmed.
v1.1.7 2026-06-20
  • Added temporary Zoom Pop-out effect: clicking/flipping a card zooms it in by 28% for 1 second before settling back.
  • Tightened screen margins and grid layout gaps on mobile devices (body padding: 4px, container padding: 8px, grid gap: 6px) to maximize card dimensions.
v1.1.6 2026-06-20
  • Fixed mobile layout margins where large white borders compressed inner images and words.
  • Reduced card-front padding to 4px (2px on mobile) and scaled card-word box to 100% of card size.
  • Increased cartoon image size constraints to cover 90% of card space.
  • Thinned mobile card borders to 2px.
v1.1.5 2026-06-20
  • Fixed Firefox grid layout bug where cards containing large images would stretch and display as different sizes than text cards. Lock container sizes via explicit relative-absolute constraints.
v1.1.4 2026-06-20
  • Fixed mobile/Android garbage collection bug where the SpeechSynthesisUtterance object was aggressively cleaned up by the engine, cutting off speech before it played. Saved active utterances to a persistent global list (`window.activeUtterances`).
  • Explicitly defined `utterance.lang = 'en-US'` to ensure consistent system-level Text-To-Speech routing on Android Chrome.
v1.1.3 2026-06-20
  • Disabled standard `.cancel()` calls on mobile browsers to prevent the Google TTS engine queue from freezing.
  • Forced fallback to default local English system voice on mobile, bypassing cloud voice downloads.
  • Added `window.speechSynthesis.resume()` before speaking.
v1.1.2 2026-06-19
  • Added HTTP Cache-Control meta tags to force browser cache clearing on page refresh.
  • Removed `max-width: 100%` restriction from `.card-word span` to allow overflow measurement for the auto-sizing font algorithm.
v1.1.1 2026-06-19
  • Added a cute Start Page overlay with version history panel.
  • Wrapped card words in a nested inline span to strictly disable browser text breaks.
  • Improved font-size recalculation to fit words on exactly one line.
v1.0.0 2026-06-19
  • Initial release of the 5x4 HTML5 Memory Game.
  • Speech synthesis voice playback when clicking animal cartoon cards.
  • No voice playback when clicking written word cards.
  • Synthesized Audio effects (flip pop, match chime, mismatch buzz, win fanfare).
  • Confetti party on matching all cards.
